# 🌾 Paddy Leaf Disease Classification using PyTorch

## 📌 Overview
This project focuses on classifying paddy leaf diseases using a deep learning model built with **PyTorch**. The model analyzes leaf images and predicts the type of disease affecting the paddy crop. The goal is to assist in early detection and improve agricultural productivity.

---

## 🚀 Features
- Image-based disease classification
- Deep learning model using PyTorch
- Preprocessed dataset (resized & normalized images)
- CNN-based architecture for feature extraction
- Notebook-based implementation (no TensorFlow used)

---

## 🧠 Tech Stack
- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib
- Jupyter Notebook
